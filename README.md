### BubbleBubble
'보글보글 방울 소동' 이라는 제목에 맞게 스펀지밥을 배경으로 사용자와 상호작용 하는 모바일 웹입니다.
- 사용 방법
1. 사용자 클릭/터치 위치에 랜덤 크기의 비눗방울 생성
2. 모션 센서를 통해 디바이스를 빠르게 흔들면 효과음과 함께 비눗방울 터트리기
3. 오리엔테이션 센서를 통해 사용자가 디바이스를 기울이는 방향으로 비눗방울 움직이기
